# Deposition Digest Landing Page

A simple, non-commercial landing page for Deposition Digest - a tool designed to assist paralegals in summarizing legal deposition transcripts.

## Features

- Clean, modern design
- Responsive layout for all devices
- Professional color scheme appropriate for legal services
- Sections covering features, how it works, and about information

## Files

- `index.html` - Main HTML structure
- `styles.css` - Styling and layout
- `README.md` - This file

## Usage

Simply open `index.html` in a web browser to view the landing page. No build process or dependencies required.

## Customization

You can easily customize:
- Colors in the `:root` CSS variables
- Content in the HTML sections
- Layout and spacing in the CSS file
